Images attendues pour la page Station F (/station-f)
======================================================

Dépose simplement les fichiers ci-dessous dans ce dossier
(public/stationf/). La page les chargera automatiquement.

Fichiers à fournir :

  - logo.png         Logo officiel Station F (PNG ou SVG, fond transparent
                     conseillé). Affiché en haut de la page à côté du
                     logo AdSync.

  - cover.jpg        Image hero principale (paysage, ~1600x900).
                     Photo du campus Station F, façade, etc.

  - campus.jpg       Vue intérieure du campus (~1200x800).

  - fighters.jpg     Photo du Fighters Program / équipe / event
                     (~1200x800).

  - logo.svg         (Optionnel) version vectorielle du logo,
                     prioritaire sur logo.png si présent.

Tant que ces images ne sont pas déposées, la page affichera
des emplacements gris stylisés pour conserver une mise en page
propre.
